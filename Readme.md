## Тестовое задание для вакансии Фронтенд-разработчик в компанию Исходный код.

#### Автор: Петр Кириенко

### Запуск:
* установить зависимости
* 'npm run build' создает продакшн версию проекта
* 'npm run start' запускает web сервер в режиме разработки


### Инструменты
* HTML/CSS
* React.js
* сборка на Webpack
